# How to Add Product Images (Free Methods)

## Option 1: Google Drive (Recommended)

1. **Upload image to Google Drive**
   - Go to https://drive.google.com
   - Upload your product image

2. **Get shareable link**
   - Right-click the image → "Get link"
   - Change to "Anyone with the link can view"
   - Copy the link (looks like: `https://drive.google.com/file/d/FILE_ID/view?usp=sharing`)

3. **Convert to direct image URL**
   - Extract the FILE_ID from the link
   - Use this format: `https://drive.google.com/uc?export=view&id=FILE_ID`
   
   **Example:**
   - Original: `https://drive.google.com/file/d/1ABC123xyz/view?usp=sharing`
   - Direct URL: `https://drive.google.com/uc?export=view&id=1ABC123xyz`

4. **Paste in admin panel**
   - Go to Admin → Add/Edit Product
   - Paste the direct URL in "Image URL" field

## Option 2: Imgur (Free, No Account Required)

1. **Upload to Imgur**
   - Go to https://imgur.com/upload
   - Drag & drop your image (no account needed)

2. **Get direct link**
   - After upload, right-click the image → "Copy image address"
   - URL will look like: `https://i.imgur.com/ABC123.jpg`

3. **Paste in admin panel**
   - Use this URL directly in the "Image URL" field

## Option 3: Unsplash (Free Stock Photos)

1. Go to https://unsplash.com
2. Search for product-related images
3. Click image → "Copy image URL"
4. Paste in admin panel

## Tips

- **Image size**: Keep images under 2MB for fast loading
- **Aspect ratio**: 4:3 works best (e.g., 800x600px)
- **Format**: JPG or PNG
- **Leave blank**: If you don't add an image URL, a default category image will be used

## Testing Your Image URL

Before saving, paste the URL in a new browser tab. If the image displays, it will work!
